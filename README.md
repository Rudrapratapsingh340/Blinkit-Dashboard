# Blinkit-Dashboard

# 1. Project Title / Headline
Blinkit Analytics: Sales & Outlet Insights Dashboard

A dynamic and interactive data visualization dashboard built using Power BI to analyze Blinkit’s sales performance, outlet distribution, and product trends across different regions and categories.


# 2. Short Description / Purpose
This Power BI dashboard provides a comprehensive overview of Blinkit's sales data, including total sales, item categories, outlet performance, and customer ratings. It helps users identify trends, compare outlet types, and make data-driven business decisions.


# 3. Tech Stack
The dashboard was built using the following tools and technologies:

Power BI Desktop – Main data visualization platform

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Calculated measures and KPIs

Data Modeling – Relationship building and data structuring

# 5. Key Features
📈 Total Sales Overview – Displays overall revenue ($1.20M)

🛒 Item Analysis – Breakdown by item types (Fruits, Snacks, Dairy, etc.)

🏪 Outlet Insights – Performance by outlet size, type, and location

🌍 Tier-wise Analysis – Sales comparison across Tier 1, 2, and 3 cities

⭐ Customer Metrics – Average rating and item count

📊 Interactive Filters – Filter by outlet size, location, and item type


# 6. Dashboard KPIs
Total Sales: $1.20M

Average Sales: $141

Number of Items: 8523

Average Rating: 3.9


# 7. Insights Gained
Tier 3 cities contribute the highest sales
Supermarket Type 1 generates maximum revenue
Fruits and snack foods are top-performing categories
Medium-sized outlets dominate sales distribution
